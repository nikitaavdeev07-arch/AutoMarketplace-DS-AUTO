# 🚗 DS AUTO — Веб-интерфейс автомобильного маркетплейса

Курсовой проект по дисциплине «Информационные технологии и программирование».  
Разработка клиентского веб-приложения автомобильного маркетплейса включающая в себя удобную навигацию, адаптивный интерфейс и подробную информацию о каждом автомобиле.

# 📋 Техническое задание

## Общие требования

Назначением курсовой работы является разработка современного адаптивного веб-интерфейса для автомобильного маркетплейса, обеспечивающего удобное взаимодействие пользователя с каталогом автомобилей и функциональными возможностями сайта. 

Сайт предоставляет пользователю возможность просмотра каталога автомобилей удобной и интуитивно понятной для каждого, а также отправки пользовательской заявки через форму обратной связи.

В процессе разработки были применены современные технологии веб-разработки: HTML5, CSS3 и JavaScript.

---

## Функциональные требования

### Обязательные:
- реализация страниц по брендам автомобилей;
- просмотр карточек автомобилей с их подробными характеристиками;
- оформление заявки через форму обратной связи;
- адаптивное отображение интерфейса;
- валидация формы;


### Желательные:
- реализация анимационных эффектов;
- динамическое обновление интерфейса.

---

## Технические требования

Сайт функционирует на современных компьютерах и мобильных устройствах. Минимальные технические характеристики: процессор от 1.6 ГГц, ОЗУ от 512 МБ. Рекомендуемое разрешение экрана — от 1024×768 пикселей. 

---

## Программные требования

Сайт работает в современных браузерах:
- Google Chrome;
- Mozilla Firefox;
- Microsoft Edge;
- Safari;
- Yandex;

Разработка выполнялась в Visual Studio Code с использованием Git и GitHub. Публикация проекта осуществляется через GitHub Pages.
